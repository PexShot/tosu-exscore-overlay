# tosu-exscore-overlay

A simple configurable overlay for tosu that mimics IIDX EX scoring (osu!mania recommended)



To edit the awarded point value you may import or directly configure config.json



The default value assigned as of v1.1 are:

300g = 2

300 = 1

200 = 0

100 = 0

50 = 0

Miss = 0



(This is the value of the original IIDX EX scoring)



works out of box with tosu v4.19.2 with osu!lazer, compatibility with other versions is currently unknown



NOTE: changes to config.json wont be shown until you restart the overlay
