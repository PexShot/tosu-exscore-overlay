To import your custom point values you can replace the config.json in this folder with your own or edit it directly.

the config.json must follow the same structure as the original to avoid errors.

The default config.json structure:

{
  "_comment" : "PFg is MAX(300g), PF is Perfect(300), GR is Great(200), GD is Good(100), BD is Bad(50), MS is Miss(0).", 
  "presetname": "IIDX Style Preset (default)",
  "PFg" : 2,
  "PF" : 1,
  "GR" : 0,
  "GD" : 0,
  "BD" : 0,
  "MS" : 0
}

if you are importing make sure the file is always named config.json
